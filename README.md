## Project Overview  
**DecodeX** is a web-based Employee Management System built using PHP (and related technologies) which allows administrators and managers to manage employee data, roles, and operations.

### Key Features  
- Create, read, update, and delete operations for employees  
- Role-based access control (e.g., Admin, Manager, Employee)  
- Departments, designations, and employee profiles  
- Login/Signup authentication & session management  
- Reports generation (employee list, department-wise, etc)  
- Responsive UI 
- MySQL database integration  

## Tech Stack  
- **Backend**: PHP  
- **Database**: MySQL (or MariaDB)  
- **Frontend**: HTML5, CSS3, JavaScript  
- **Version Control**: Git & GitHub
- **Server**: Apache
